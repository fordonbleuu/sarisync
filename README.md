# SariSync POS

SariSync is a modern, mobile-first Point of Sale (POS) and inventory management system designed specifically for Sari-Sari stores and small retail businesses. It helps store owners track sales, manage inventory, monitor debts, and analyze financial performance with ease.

## 🚀 Features

- **Point of Sale (POS)**: Quick and intuitive checkout process with support for discounts and multiple payment methods (Cash and "Utang"/Credit).
- **Inventory Management**: Track stock levels in real-time. Get alerts for low stock and out-of-stock items.
- **Debt Ledger**: Manage customer "Utang" efficiently. Track partial payments and set due date reminders.
- **Expense Tracking**: Record store expenses to get an accurate view of your net profit.
- **Audit Reports**: Daily financial summaries showing gross revenue, COGS (Cost of Goods Sold), expenses, and net margins.
- **Offline First**: Works completely offline using a local SQLite database (powered by Drift).
- **Visual Design**: Modern and clean UI with a dedicated design system for a professional look and feel.

## 🏁 Getting Started

### Prerequisites

- Flutter SDK (^3.11.5)
- Dart SDK

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/sarisync.git
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
